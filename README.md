# Apple Evolution: The Scrollytelling Experience

A premium, interactive web experience chronicling the evolutionary journey of an apple. This project transforms a series of static educational pages into a high-performance React application featuring advanced scrollytelling, immersive animations, and real-time biological data analysis.

## ‚ú® Features

- **Immersive Scrollytelling**: Seamless transitions between four evolutionary stages (Origin, Bloom, Harvest, Essence) controlled by the user's scroll.
- **Nature-Tech Aesthetic**: A fusion of organic imagery and futuristic UI elements using Glassmorphism and specialized typography.
- **Interactive Bio-Metrics**: Real-time biological data modals and progress tracking indicators.
- **High Performance**: Built with Vite and React for lightning-fast loads and smooth 60fps animations via Framer Motion.
- **Fully Responsive**: Optimized for all device sizes, from mobile to ultra-wide displays.

## üöÄ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shishir-sh26/Apple_origin.git
   cd stitch
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

## üì¶ Deployment

To build the project for production:

```bash
npm run build
```
